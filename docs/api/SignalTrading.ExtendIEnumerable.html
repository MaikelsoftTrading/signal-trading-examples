Class ExtendIEnumerable

The framework generates signals from Pricing data that is paired with additional data that must be analyzed. The System.Collections.Generic.IEnumerable<T> interface is therefore extended with methods that transform single elements to (Pricing, TData) tuples, and a method that generates signals from these tuples.
